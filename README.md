# Metodes_Numerics_II
Estaria bé anar apuntant aquí el que feu cad dia. Podeu apuntar com a títul les paraules claus del que heu fet i la data i com a text un breu resum del que heu fet.

## Comandes

    git clone "enllaç" : copia un repositori (no oblidis abans, des de la terminal, anar a la carpeta on vols que el creei).
    git branch : et diu a quina branca estàs (també surten la resta de branques, crec).
    git branch checkout "Nom": et mous de la branca on ets actualment a la branca "Nom".
    git branch -d "Nom": borra la branca "Nom" (molt de compte amb això, en princpi no ho   heu de fer servir, és bastant perillós, el que es borra no es pot recuperar).
